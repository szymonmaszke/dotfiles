**dotfiles**
=========

My current Arch Linux setup files, main treats:

	- keyboard oriented (zsh, neovim, i3-gaps, firefox, tmux, ...)
	- minimalistic and fast
	- everything kept in the same sigle grayish tone
	- automation (aliases, scripts)
